# Breadcrumb 面包屑

## 使用和示例

### 基础用法
:::demo
```html
<div class="doc-row">
  <h5 class="doc-row__title">默认样式</h5>
  <div class="doc-row__body">
    <ul class="nv-breadcrumb">
      <li class="nv-breadcrumb__item">
        <a>一级菜单</a>
      </li>
      <li class="nv-breadcrumb__item">
        <a>二级菜单</a>
      </li>
      <li class="nv-breadcrumb__item">三级菜单</li>  
    </ul>  
  </div>
</div>
```
:::


## API
### 类名

| Class  |  Description  |
|---|---|
| `nv-breadcrumb`  | 基础类名  |
| `nv-breadcrumb__item`  | 导航项  |
